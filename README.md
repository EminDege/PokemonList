# Pokémon List App

A simple React application that fetches Pokémon data from a GraphQL API and displays a list of Pokémon names with various interactive features, including filtering by name and displaying detailed information about the selected Pokémon.

## Features

- **Display Pokémon**: Fetches and shows a list of Pokémon, displaying their names, height, weight, and base experience.
- **Search Filter**: Allows users to search for Pokémon by name, with results updated dynamically as the filter text changes.
- **Adjustable Display Count**: Users can control the number of Pokémon displayed on the page.
- **Pokemon Details**: Clicking on a Pokémon's name shows more detailed information such as height, weight, and base experience.

## Tech Stack

- **React**: JavaScript library for building the user interface.
- **GraphQL**: Query language for APIs used to fetch the Pokémon data.
- **Apollo Client**: A GraphQL client to interact with the API.
- **CSS**: For styling the application.
